# 🍕 Food Donation Platform

A full-stack web application connecting food donors with recipients to reduce food waste and fight hunger.

## 🚀 Features

- Food donation listing
- Real-time requests
- User authentication
- Beautiful UI/UX
- Responsive design

## 🛠️ Tech Stack

- **Frontend:** React, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Real-time:** Socket.io

## 📦 Installation

```bash
# Backend
cd backend
npm install
npm start

# Frontend  
cd frontend
npm install
npm run dev
